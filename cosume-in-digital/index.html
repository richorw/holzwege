我的数字消费

2020-03-08

今天打开 bobby 小松鼠（一款帮助用户整理定期订阅账单的 app），惊觉每个月要花 ¥265 在各种数字消费上。最近也一直在家「消费」自己购买的课程和流媒体，所以不妨分享一下自己目前订阅、以及订阅过的服务，以总结自己在二次元世界作出的微小贡献。

娱乐类

Spotify（日区，学生优惠）

480日元_月，折合人民币32元_月

本来是忠实的虾米会员，但身在日本，国内的流媒体因为版权问题大多不能播放，所以考虑了 Apple Music 和 Spotify。听的音乐主要包括日本流行音乐、后摇、爵士等等，而 Spotify 日区的歌还是比较全的，比如在 Apple Music 上找不到的 Euphoria 在 Spotify 上就有。以及可以配合家里的 Google Home Mini 傻儿子食用，大喊一声「Hey Google, play Spotify!」有奇效。最后有学生优惠，一个月一杯奶茶的价格，你值得拥有。

Amazon Prime（日区，学生优惠）

1900日元_月，折合人民币10元_月

个人感觉日本留学生的一大利器就是日亚学生会员，不但免运费，买书积分多，还有 Prime Video 可以看。虽然片源比较古老，无法和 Netflix 相比，但每季新番的一两个主打还是会上的，补旧番和老电影也颇为实用。

生产力工具

Cargo

13美金_月，折合人民币90_月

Cargo 是一个建站工具，目前自己的设计作品集就 host 在上面。使用体验还不错，模板简洁漂亮，艺术性颇强，自定义工具和编程支持都很好，唯一的缺点就是贵了些。Cargo 其实也有免费的版本，若干年前的摄影作品集 host 在上面。不过付费版本没有容量限制、可以绑定域名、可以编辑源代码，自定义程度会高很多。

Notion（个人）

48美金_年，折合人民币28_月

后知后觉今年才入 Notion 的坑，主要用来个人知识体系管理和内容协作。其实有学生免费套餐，但不想绑定学校邮箱，加之心甘情愿付些钱，所以选择了个人套餐。

Bear

人民币103/年

自从 Bear 问世以来就一直把它当作主力 Markdown 编辑器（你所看到的这篇文章自豪地由 Bear 编辑器编写完成），优雅的界面和标签功能一直都很好用，去年绝大部分的文章都出自于此。即使有了 Notion，应该也会继续订阅。

Adobe / Sketch / Figma / FramerX

0/月

从大学时代开始养成了非正版软件不用的习惯（尤其是 Adobe，高中时还是杰出的盗版软件安装大师），于是大三第一年来日本的时候开始订阅 Adobe 全家桶学生套餐，每年大约 3000 日元，回国也持续用着。幸好后来重新考上了美术大学，可以有免费的 Adobe 使用，于是放肆下载。Sketch 当时买断了一个版本，现在改成了订阅制也没有重新付费，但仍然可以使用。之后工作也是设计职，妈妈再也不用担心我的设计软件了。

教程类

Designcode

54美金_年，折合人民币32_月

去年底趁黑五打折，订阅了一年Meng出品的 Designcode 教程系列，给设计师看的编程教程，总体来说还是非常值回票价的。目前看完了 CSS、Vue、Figma 和 FramerX 的部分，他的总体思路都是从具体案例（Designcode 的设计本身）出发，对掌握软件和基础编程技术来说比较有用，但不提供设计思路和编程思路的讲解，个人觉得不仅适合设计师快速了解编程的基本方法，也适合程序员快速学习时下流行的 Figma 和 FramerX 等软件操作。不过要真想学编程，还得借助别的教程。

Frontend Masters（GitHub Student）

0/6个月

如标题所示，是前端学习的网站。朋友推荐了上面的 Vue 教程，于是先花了$39订阅了一个月，某天 google 的时候无意中发现了它是 GitHub Student 豪华套餐的一员，六个月免费学习。看到就是赚到，嗯。

Learn JavaScript Online

$29

一个交互性非常强的 JavaScript 入门课程，作者答疑也非常快，好评。

Rosetta Stone（美区）

40美金_3个月，折合人民币92_月

二月下定决心开始学德语（此生目标是至少学会德意日，目前完成度33.3%），问了擅长德语的朋友学习方法，朋友推荐了经典的 Rosetta Stone，火速订阅。web 端和移动端 app 支持都挺好的，用了一个月下来觉得比较适合入门，但不适合深度学习，尤其是语法部分，目前仍然很混沌。所以目前打算是学完三个月的混沌德语入门，再进入语法进阶。

3D for Designers

黑五价格忘记了，目前是$99 还没学，且不评。只记得黑五价格很便宜！（雾
